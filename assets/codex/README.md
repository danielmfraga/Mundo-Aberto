# Imagens do Codex

Coloque aqui os retratos das criaturas. O `codex.html` referencia por nome de arquivo.

| Criatura | Arquivo esperado |
|---|---|
| Fantasma Sofredor | `fantasma-sofredor.jpg` |

Se o arquivo não existir, o card mostra um placeholder elegante automaticamente
(o `<img>` se remove sozinho via `onerror` e a silhueta espectral aparece).

Formato sugerido: JPG/WEBP, proporção ~4:5 (retrato), ~800px de largura.
